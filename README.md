# 🎮 StressLess Ghub 228

## 📖 Description

StressLess Ghub 228 is a mobile application designed to help users relax and reduce stress. The app serves as a game hub that provides simple and enjoyable games intended to help lower cortisol levels and improve mood. By offering engaging and fun activities, the app aims to give users a quick and accessible way to relieve stress anytime.

## 🕹️ Available Games

The app currently includes several mini-games:

🎯 Tap Rush – Tap the button as fast as you can before the timer runs out.

🎨 Simon Says – Memorize and repeat the sequence of colors.

❓ Quick Quiz – Answer fun trivia questions about mobile games.

🧠 Number Guess – Guess the secret number using hints before time runs out.

## 📱 Requirements

Make sure you have the following installed:

Node.js

Expo CLI

Expo Go mobile application

## ⚙️ Installation

Follow these steps to install the dependencies and run the app:

1. Clone the repository
2. Go to the project folder
3. Install the dependencies (type "npm install" in terminal if you are using vscode)
4. Run the app (type "npx expo start" in terminal)
5. Open the app using **Expo Go** in your phone and scan the QR code.
6. Now, you're all set up!
